# 像素珠转换器 - 静态版本使用说明

## 简介
这是一个完全静态的像素珠图案转换器，无需安装任何软件或依赖，直接在浏览器中打开即可使用。

## 使用方法
你只需要关注 `out` 文件夹，请将 `out` 文件夹复制到您的电脑上

### 方法一：直接打开（推荐）
1. 进入 `out` 文件夹
2. 双击打开 `index.html` 文件
3. 应用将在您的默认浏览器中打开

### 方法二：通过本地服务器打开
**方法的前置条件：** 在电脑中已安装 python
如果您遇到跨域问题，可以使用以下方法：

**Windows用户：**
1. 打开命令提示符（cmd）
2. 进入 `out` 文件夹所在目录
3. 输入命令：`python -m http.server 8000`
4. 在浏览器中访问 `http://localhost:8000`

**Mac用户：**
1. 打开终端
2. 进入 `out` 文件夹所在目录
3. 输入命令：`python3 -m http.server 8000`
4. 在浏览器中访问 `http://localhost:8000`

## 功能说明

### 主要功能
- 📷 **图片上传**：支持拖拽或点击上传图片
- 🎨 **像素转换**：将图片转换为像素珠图案
- 🎯 **图案编辑**：支持手动修改颜色和图案
- 📊 **统计信息**：显示所需各种颜色珠子的数量
- 📄 **导出功能**：支持导出为PDF和PNG格式

### 操作指南
1. **上传图片**：在左侧面板上传您的图片
2. **设置画板尺寸**：调整像素珠图案的宽度和高度
3. **生成图案**：点击"生成拼豆图案"按钮
4. **编辑图案**：使用画笔工具修改图案
5. **导出结果**：选择PDF或PNG格式导出

### 快捷键
- `Ctrl+Z` / `Cmd+Z`：撤销
- `Ctrl+Y` / `Cmd+Y`：重做
- `V`：选择工具
- `B`：画笔工具
- 鼠标滚轮：缩放图案
- `Alt+拖拽`：平移图案

## 注意事项

1. **浏览器要求**：建议使用Chrome、Firefox、Edge等现代浏览器
2. **图片格式**：支持JPG、PNG等常见图片格式
3. **文件大小**：建议图片大小不超过5MB
4. **离线使用**：所有功能都在浏览器中运行，无需网络连接
